# practicum_dle_sprint_4

# Описание проекта

В проекте реализовано
- создание и обучение Мультимодальной нейросети для предсказания калорийности блюда по его фотографии, ингредиентов (текст) и массе (число);
- расчет метрик на тестовом датасете и анализ получившихся результатов;

#  Инструкция по развертыванию проекта

## Клонируйте репозиторий

```
git clone https://github.com/russele7/practicum_dle_sprint_4.git
```

## Перейдите в папку репозитория

```
cd practicum_dle_sprint_4
```

## Создайте виртуальное окружение

```
py -3.10 -m venv venv
```

## Активируйте виртуальное окружение

```
source venv/Scripts/activate
```

## Установите необходимые зависимости

```
pip install -r requirements.txt
```

## Запустите в созданном окружении ноутбук solution.ipynb с реализованным пайплайном
